To rename or move the files from one folder to another use git mv command so that github tracks it as renamed and not many changes
```git
git mv index.html 01_Basics/
```
Windows file system is case-insensitive. So for similar names Rename through a temporary name.

```
git mv 03LoadingIcon tempFolder
git mv tempFolder 03loadingIcon
```

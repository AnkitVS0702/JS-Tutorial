## Inherit
inherit is a special CSS keyword.

It tells property to: copy computed value from parent

eg
```
border-radius:inherit;
font: inherit;
```

Some property like
color,
font-size,
font-family,,
line-height,
text-align
are inherited by default

But some like 
width,
height,
margin,
padding,
border,
background,
border-radius
are not  !!
